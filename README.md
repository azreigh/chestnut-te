# Chestnut TE
Basic modal text editor made using the [Brux](https://github.com/KelvinShadewing/brux-gdk) game engine.
## Dependencies
* Brux v.0.2.11 or above
* [Hazel](https://github.com/tulip-sudo/hazel)
